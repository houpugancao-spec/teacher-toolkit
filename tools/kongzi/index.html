6孔子小档案 Fact file

认识真实的孔子 The real Confucius

👤
孔子 Kǒngzǐ — 英文叫 Confucius，中国最有名的老师。
⏳
约 2500 年前 的人（春秋时期）。
📏
身高约 1.86 米，高大健壮，人称「长人」。
😢
身世很惨：3 岁失去父亲，17 岁失去母亲，家里很穷。
📚
靠 自学 成才，后来成为老师，被尊为「圣人」(shèngrén, a great sage)。

7说一说 Speak
